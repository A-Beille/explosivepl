<h1>ExplosivePL</h1>
<h2>Plugin d'alertes à la bombe</h2>
<p>Ce plugin rend Minecraft plus réaliste, en rajoutant des alertes à la bombe toutes les 5 minutes dans le serveur, et en expulsant tous les membres. Un pur plaisir !</p>
